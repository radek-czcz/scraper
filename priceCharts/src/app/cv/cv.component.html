RÉSUMÉ

Hi, my name is Radosław Czyżowicz, I’m 39 years old. I have 3 kids in the age range 3-11.

My education and my job is focused around mechanical engineering. I have bachelor degree in this field and I work for a company from the automotive industry.

To be concise, my current daily work routine is taking care of car seats products, from the construction, documentation point of view. Currently I’ve been working about 10 years in the aforementioned position and I feel it has come time to change.

About 5 years ago I have began my adventure in coding. I started with writing VBA macros for CAD software and for Excel. I always liked mathematical puzzles, where one must be strictly tied to the prevalent rules, to successfully find the solution of the problem… so I quickly went to the next level and started learning java with some of my small private project.

Currently one of my main hobbies is widening my coding skills. Last 2 years I was dedicating much of my free time to learn some of JavaScript and Angular. Recent 2 months I wanted to make my first webpage. It’s nothing big, but from my point of view, for a self-learner, it’s some kind of milestone. This site is only for demonstration, if someone would want to know, what I have learnt so far.

If you are interested in the content of the webpage and how I have made it, please read the second page of my resume. I have prepared short description of the frameworks and libraries, which I have used.

Below you can find informations about the site which I have prepared for recruiters. This informations could give you a short description for my skills. The source code for my web application can be found on GitHub: radek-czcz/scraper (github.com)

The site is to be found on url http://srv59554.seohost.com.pl/. Be sure to not include “s” after “http” in the url string, as then the frontend will not be able to obtain backend data because of CORS restrictions. All the server setup on the seohost servers i have done myself about month ago. I have learnt to handle with FTP file tool FileZilla to transfer my files to my hosting. I have gained access to my NodeJS application through terminal with tool PuTTY. I have there established a MySQL-Database to be able to communicate with my backend app and to serve the data to frontend application written in Angular.

All the data, which are to be found in database, come from scraping scripts written in Puppeteer. The site doesn’t update by itself, but i can update it from my local computer, by running these scripts.

The site presents charts with prices for a couple of product categories, which i choosed at the beginning. The charts are displayed thanks to Chart.js open source library. Backend gets the data from DB with the use of mysql2 npm library. The data is being served with the backend server Node application built on Hapi framework. To be able to use modern JavaScript syntax I have put a babel-node transpiler in my dependencies.

As it has been mentioned I have built my frontend with the use of Angular and TypeScript. Visible components of website come from Angular-Material library.
